Changelog

v0.1.4 — Korean onboarding

Made first-run onboarding Korean-first with a 한국어/English selector, localized permission/model/access guidance, and Korean hosted/BYOK key-entry copy.

v0.1.3 — Microphone capture fix

Packaged the AVAudioEngine microphone sidecar into the signed Mac app so first-run/reboot installs capture the user microphone for STT, and made recording permission errors name the exact missing macOS permission.

v0.1.2 — Hosted beta access

Added Hosted beta token onboarding, BYOK fallback, relay token verification, and hosted Soniox/Upstage routing for transcription, translation, and final notes.

v0.1.1 — Visual refresh

Replaced the landing mockup with a fitted real app screenshot and refreshed the app icon to a flatter white writing mark across web and macOS assets.

v0.1.0 — Public beta

Signed and notarized universal macOS DMG, first-run permission onboarding, Soniox/Upstage BYOK setup, local meeting library, and signed Tauri updater feed.
