Changelog

v0.1.9받아쓰기 사용량 측정최신

받아쓰기 사용 시간을 정확히 기록하도록 내부 미터링을 추가했습니다(향후 사용량 기반 과금 대비). 기능 동작은 0.1.8과 동일합니다.

v0.1.8받아쓰기 안정화 + 깃펜 리브랜드

받아쓰기 연속 사용 시 전사가 비던 문제를 고치고(STT 종료 순서 수정), 첫·끝 단어 잘림을 보정했습니다. 받아쓰기 오버레이를 양피지 깃펜 디자인으로 새로 만들고, 앱·메뉴바 아이콘을 깃펜으로 리브랜드했습니다. heybartleby.com도 미팅 노트 + 받아쓰기 메시지로 리뉴얼했습니다.

v0.1.7Dictation + in-app feedback

New: hold the Fn key anywhere to dictate — your speech is typed straight into the focused app, and every dictation is saved in a new dictation list. Enable it in Settings, Recording (needs macOS Accessibility permission). Added in-app feedback in Settings, About. Removed the live English to Korean transcript translation to focus on clean Korean notes; meeting recording and Korean summaries are unchanged.

v0.1.6In-app updates + anonymous usage

Updates now appear in-app with release notes and a progress bar instead of a system dialog, including support for required updates. Bartleby also records anonymous install and usage counts (a hashed machine id and plain numbers only, never meeting content) to guide future improvements; see the privacy page.

v0.1.5Usage analytics with opt-out

Bartleby now records anonymous usage events (app opens, recording start/stop, note generation) to guide improvements. Meeting audio, transcripts, summaries, titles, and API keys are never sent, and you can turn analytics off anytime in Settings.

v0.1.4Korean onboarding

Made first-run onboarding Korean-first with a 한국어/English selector, localized permission and model guidance, and Korean access copy.

v0.1.3Microphone capture fix

Packaged the AVAudioEngine microphone sidecar into the signed Mac app so first-run/reboot installs capture the user microphone for STT, and made recording permission errors name the exact missing macOS permission.

v0.1.2Hosted beta access

Added Hosted token onboarding, relay token verification, and hosted Soniox/Upstage routing for transcription and final notes.

v0.1.1Visual refresh

Replaced the landing mockup with a fitted real app screenshot and refreshed the app icon to a flatter white writing mark across web and macOS assets.

v0.1.0Public beta

Signed and notarized universal macOS DMG, first-run permission onboarding, hosted Soniox/Upstage setup, local meeting library, and signed Tauri updater feed.
