하나의 한국어 STT 엔진, 두 가지 쓰임

말하면, 글이 됩니다

Bartleby는 동일한 실시간 한국어 음성 인식 파이프라인 위에서 두 가지 일을 합니다. 받아쓰기와 회의 노트, 둘 다 일급 기능입니다.

받아쓰기 · Dictation

Fn 키를 누른 채 말하면, 커서에 들어옵니다

이메일, 메신저, 코드 에디터, 노션 — 텍스트를 입력할 수 있는 곳이라면 어디서든. 손을 키보드에서 떼지 않고 생각의 속도로 적습니다.

  • 1🌐(Fn) 키를 누른 채 자연스럽게 말합니다.
  • 2키를 떼면 인식된 문장이 바로 커서 위치에 삽입됩니다.
  • 3한국어도, "이 API를 deprecate 해줘" 같은 한·영 혼용도 매끄럽게.

왜 Bartleby인가

한국어를 제대로 듣습니다

번역 도구가 아닙니다. 한국어와 한·영 혼용을 정확히 받아쓰는 것이 우리의 무기입니다.

한국어 · 한영 혼용에 강함

다른 받아쓰기 도구가 약한 한국어와 코드 스위칭을, 회의 용어와 함께 한 세션 안에서 정확하게 인식합니다.

설정 없는 Hosted

키 입력도, 별도 계정도 필요 없습니다. Notique가 Soniox·Upstage 비용을 처리하니 받아쓰기에만 집중하세요.

기록은 내 기기에

녹음과 받아쓰기 기록은 사용자 Mac에 보관됩니다. 내 데이터는 내 손 안에.

실시간 인식

Soniox stt-rt-v4 엔진으로 말과 거의 동시에 글자가 따라옵니다.

한국어 요약 엔진

Upstage solar-pro3가 회의의 맥락을 살린 깔끔한 한국어 요약을 만듭니다.

조용한 자동 업데이트

서명·공증된 빌드로 백그라운드에서 매끄럽게 업데이트됩니다.

지금, 받아쓰기를 시작하세요
