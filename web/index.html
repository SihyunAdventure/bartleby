영어 회의를,
한국어 노트로.

마이크와 시스템 오디오를 함께 캡처하고, Soniox 실시간 전사를 Upstage Solar Pro 한국어 회의록으로 정리합니다. 모델 선택은 단순하게, 키는 내 계정으로.

회의 녹음 앱의
복잡함을 다시.

유저가 헷갈릴 선택지는 줄이고, 실제로 필요한 설치·권한·BYOK 흐름만 첫 실행에서 끝냅니다.

01

권한을 단계별로

마이크와 화면 기록 권한을 한 번에 몰아 묻지 않고, 왜 필요한지 설명한 뒤 macOS 설정으로 안내합니다.

02

모델 선택 없음

현재 릴리즈는 Soniox stt-rt-v4와 Upstage solar-pro3만 사용합니다. 로컬 모델 다운로드나 불필요한 LLM 옵션은 없습니다.

03

BYOK by design

Soniox와 Upstage API 키를 온보딩에서 검증하고 macOS Keychain에 저장합니다. 사용량 비용은 제공자 계정으로 직접 관리합니다.

04

실시간 전사

회의 중 영어 발화를 바로 받아 적고, 세션별 오디오와 transcript를 로컬 라이브러리에 보관합니다.

05

한국어 회의록

결정사항, 액션아이템, 인용을 한국어 중심으로 정리해 회의 직후 바로 공유 가능한 형태로 만듭니다.

06

조용한 업데이트

서명된 Tauri updater feed로 다음 버전을 받을 수 있게 준비했습니다. DMG와 updater artifact는 모두 서명되어 배포됩니다.

지금 다운로드.

설치 후 첫 실행에서 권한을 열고, Soniox와 Upstage 키만 붙여넣으면 바로 녹음할 수 있습니다.
