Privacy

Local-first storage

회의 오디오, transcript, 요약 노트는 사용자의 Mac에 저장됩니다.

Provider calls

Hosted beta 모드에서는 전사와 요약 요청이 Notique의 Bartleby relay를 거쳐 Soniox와 Upstage로 전달됩니다. BYOK 모드에서는 사용자가 입력한 Soniox와 Upstage API 키로 해당 제공자 API를 직접 호출합니다. 모든 토큰과 API 키는 macOS Keychain에 저장됩니다.

Usage analytics

제품 개선을 위해 익명 사용 통계를 PostHog(미국 리전)로 수집합니다. 웹사이트에서는 페이지 방문과 다운로드 버튼 클릭(앱 버전 포함)을, macOS 앱에서는 앱 실행, 온보딩 완료, 녹음 시작/종료(녹음 길이 구간), 노트 생성, hosted/BYOK 모드 같은 이벤트와 익명 메타데이터만 수집합니다. 회의 오디오, transcript, 요약 내용, 회의 제목, API 키는 절대 수집하지 않습니다. macOS 앱의 Settings → Storage에서 사용 통계 전송을 언제든 끌 수 있습니다.

App registration

앱 실행 시 익명 기기 식별자(무작위 ID의 SHA-256 해시)와 앱 버전을 heybartleby.com에 전송해 설치/활성 기록을 남깁니다. 이는 초기 사용자 식별과 향후 라이선스 처리를 위한 것입니다. 녹음을 마치면 회의 내용이 아닌 횟수와 길이(초) 숫자만 함께 집계됩니다. 회의 오디오·transcript·요약·회의 제목·API 키·이메일은 전송하지 않습니다.
