## Phase 1 — Audio Capture (Week 2, 2026-05-14 → 2026-05-20)

> 가장 어려운 기술 문제. Anarlog 코드 정독해서 패턴 차용.

- [ ] macOS ScreenCaptureKit Rust binding (`screencapturekit-rs` crate 검토)
- [ ] mic 입력 캡처 (`coreaudio-rs`)
- [ ] 시스템 오디오 (Zoom·Meet·Teams) 캡처 — ScreenCaptureKit audio
- [ ] Mic + 시스템 오디오 mixing (channel-aware, mic = left, system = right or interleaved)
- [ ] Opus 32kbps 인코딩 + 로컬 .opus 저장
- [ ] 권한 요청 플로우 (Microphone, Screen Recording)
- [ ] Tauri command: `start_recording`, `stop_recording`, `get_recording_state`
- [ ] React side: ⏺ Start 버튼 → Tauri command → 사이드바 LIVE indicator 표시

**Phase 1 acceptance**: ⌘+⇧+B 로 녹음 시작/종료, .opus 파일 로컬 저장, 시스템+마이크 오디오 모두 들림.

## Phase 2 — Streaming STT + Live UI (Week 3, 2026-05-21 → 2026-05-27)

- [ ] Soniox websocket Rust client (또는 Tauri sidecar)
- [ ] 캡처된 audio chunk 를 websocket 으로 stream
- [ ] Soniox event 파싱 (partial / final, language, timestamps)
- [ ] React: Live meeting 레이아웃 (사이드바 축소 + KO|EN 좌우 split)
- [ ] Partial transcript = `--partial` 색 + italic
- [ ] Final transcript = `--fg-primary` 진한 색 + lock
- [ ] 자동 스크롤 (사용자가 위로 스크롤하면 일시 정지)
- [ ] LIVE indicator (pulsing dot + timer)
- [ ] 미팅 제목 인라인 편집 (untitled → 자동 또는 수동)
- [ ] 일시정지 / 재개 / 종료 버튼

**Phase 2 acceptance**: 본인 한국어 미팅 1시간 라이브 transcribe → 좌측 한국어 panel 실시간 표시 (partial → final 흐름).

## Phase 3 — Translation + Summarize (Week 4, 2026-05-28 → 2026-06-03)

### 실시간 번역

- [ ] 번역 결과 우측 panel 에 append
- [ ] Bartleby brand voice 시스템 프롬프트 (정중하고 절제된 톤)
- [ ] 양쪽 panel 스크롤 sync 옵션

### 미팅 종료 후 요약
- [ ] 미팅 종료 → Solar Pro 3 요약 호출
- [ ] 요약 스키마 (markdown frontmatter):

```yaml
---
title: <auto-generated>
date: 2026-05-28T14:30:00Z
duration: 47m
language: ko
participants: [김시현, ...]
tags: []
---
## TL;DR
3-5 문장 요약

## Decisions
- 결정 1
- 결정 2

## Action Items
- [ ] @sihyun: X 하기 (2026-06-01 까지)
- [ ] @kim: Y 하기

## Key Quotes
> "..." — 화자

## Full Transcript
(한국어 원본)

## Translation
(영어 번역)
```

- [ ] 사용자 편집 가능한 요약 view
- [ ] Markdown 파일로 저장 (`~/Documents/Bartleby/{YYYY-MM-DD}-{title}.md`)

**Phase 3 acceptance**: 본인 미팅 요약 결과가 *그대로 슬랙·이메일에 paste 할 만한 품질*.
